ThinkUp - Online Platform for Students & Teachers

ThinkUp is a full-stack educational web platform allowing teachers to create, manage, and monitor online courses, while students can browse, enroll, and interact. The system uses React.js for the frontend and Node.js with Express for the backend, storing data in MongoDB. It integrates an OpenAI-based AI chatbot to assist users.

Technologies Used

  • React.js (Frontend)
  • Node.js + Express (Backend)
  • MongoDB (Database)
  • OpenAI API (AI Chatbot)
  • RESTful APIs, JSON, Axios
