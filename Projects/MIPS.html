MIPS 16 - 16-bit Single-Cycle Processor

This project focuses on the **design and implementation of a 16-bit single-cycle processor** based on the MIPS architecture. The objective was to develop a simplified MIPS processor using **VHDL** and simulate its functionality.

The project includes designing the **datapath and control unit**, implementing instructions like arithmetic, logical, load/store, and branch operations. The entire system was tested through simulation tools to validate its correctness.
