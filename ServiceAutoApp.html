Service Auto Desktop App

A desktop application built using C# and WinForms following the MVP architecture. The app manages car service operations with features for authentication, role-based forms (Admin, Employee, Manager), and database communication via SQL Server using ADO.NET. The backend communicates through RESTful Web APIs.

Technologies Used

  • C# (.NET Framework 4.7.2)
  • Windows Forms (WinForms)
  • SQL Server with ADO.NET
  • MVP Architecture + REST APIs
  • UML, ERD, dbdiagram.io
